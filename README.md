Este proyecto es un dashboard desarrollado en streamlit ocupando el archivo rama_2023_05.csv.  El objetivo principal es explorar patrones temporales y relaciones entre variables mediante técnicas de análisis y visualización de datos.

El conjunto de datos contiene los promedios diarios de los registros de distintos contaminantes en la Ciudad de México desde el 2015 a 2023, entre los cuáles podemos encontrar:
- Ozono (O3)
- Dióxido de azufre (SO2)
- Óxidos de nitrógeno (NOx)
- Monóxido de carbono (CO)
- Partículas menores a 10 micrómetros (PM10)
- Partículas menores a 2.5 micrómetros (PM2.5)
- Ácido sulfhídrico (H2S)

Y el dataset fue extraido de La Red Automática de Monitoreo Atmosférico (RAMA) que es el subsistema del Sistema de Monitoreo Atmosférico de la Ciudad de México (SIMAT) que realiza mediciones continuas y permanentes minuto a minuto. El dataset se encuentra en: https://datos.cdmx.gob.mx/dataset/red-automatica-de-monitoreo-atmosferico

El dashboard permite visualizar la evolución temporal de las variables, comparar el comportamiento de las mediciones por estaciones del año, analizar similitudes mediante PCA y MDS y explorar relaciones entre variables de forma interactiva.
